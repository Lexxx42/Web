Header (вверху страницы) - содержит кнопки, гиперссылки и поле поиска.
NavBar (столбец слева) - блок содержит гиперссылки на другие страницы, в том числе и текущую на разных языках, модальное окно установки языка (шестеренка "На других языках").
Content (в центре странице) - состоит из карточек, которые содержат кнопки, гиперссылки, изображения, кебаб-меню, ненумерованные списки.
Footer (внизу страницы) - блок содержит 2 кнопки, ссылки на политику конфиденциальности, отказ от ответственности, заявление о куки и другие гиперссылки.

Хэдер
* содержит вкладку страницы и вкладку ее обсуждения, возможно между ними переключаться;
* есть возможность читать страницу, просматривать ее код и историю редактирования с помощью гиперссылок;
* содержит панель зарегистрированного пользователя и возможность зарегистрироваться путем перехода по ссылке или зайти в профиль, если такой есть у пользователя.

Контент
* начинается с панели приветствия;
    1. содержащей ссылки на справочные материалы по редактированию и написанию статей;
    2. на статью о ресурсе, на которую попал пользователь;
    3. статистику написанных страниц на языке пользователя;
    4. гиперссылки на обзор порталов и справочную информацию;
    5. кнопка Создать статью
* основной контент в виде карточек
    1. избранная статья - частичный парсинг содержания статьи, кнопка Читать, переход на страницу, содержащую все избранные статьи, кебаб-меню: кандидаты, просмотр шаблона;
    2. хорошая статья - аналогично предыдущей;
    3. последний избранный список - предыдущий избранный список, кнопка Другие избранные списки, кебаб-меню: кандидаты, просмотр шаблона;
    4. изображение дня - изображение, ссылка на статью, кебаб-меню: просмотр шаблона;
    5. из новых материалов - парсинг фактов из статей в ненумерованном списке со ссылками на статьи, кнопка Обсудить, кебаб-меню: предложения, архив, просмотр шаблона;
    6. текущие события - актуальные темы (статьи, которые часто редактируются за промежуток времени - x), парсинг информации из статей с последними мировыми новостями с ссылками на статьи, кнопка Другие текущие события, кебаб-меню: кандидаты, архив, просмотр шаблона;
    7. в этот день - текущее число и месяц текстом, ссылка на страницу текущего числа и месяца, ссылка статью, посвященную государственному празднику в эту дату, ненумерованный список знаменательных событий, которые произошли в прошлом в эту дату, кнопка Читать, кебаб-меню: просмотр шаблона;
    8. совместная работа недели - ненумерованный список статей, которые развивают сообщество и приглашают к совместному написанию пользователей, кебаб-меню: кандидаты, архив, просмотр шаблона.
* Ссылки на полезные ресурсы, которые расположены на серверах Википедии: лого, ссылка, описание. Расположение bento menu.

Панель навигации
* основные ссылки: главная страница, содержание, избранная статья, случайная статья, текущие события, пожертвовать;
* участие: все, что относится к взаимодействию с пользователем - сообщить об ошибке, форум, справка и т.д.
* инструменты: все, что нужно для написания страниц - ссылки, правки, служебные страницы и т.д.
* печать/экспорт: создать книгу, скачать в PDF, версия для печати;
* в других проектах: перечисленны другие проекты Википедии;
* На других языках:
    1. список доступных языков для текущей страницы, гиперссылки на страницу на этом языке.
    2. модальное окно Установки языка:
    * кнопки: Отмена, Применить установки
        Вкладка Отображение
            + Свичер Язык/Шрифты
            + когда включен свичер Язык: можно выбрать язык отображения
            + ссылка на страницу входа в профиль пользователя
            + когда включен свитчер Шрифты: можно активировать чекбокс Скачать шрифты при необходимости, получить дополнительную информацию по шрифтам
        Вкладка Ввод
            + отображаются доступные языки письма в виде кнопок: русский, English и кнопка ...
            + для выбранного языка отображается набор radio buttons с возможностью выбора раскладки.

Футер
